# COMPLETED 資料夾

> **用途**：存放已完成的任務（靜態記憶/壓縮，權重 50%-80%）

---

## 使用方式

1. **AI 會自動移動**：當任務完成後，從 `ACTIVE/` 移到這裡
2. **檔案命名格式**：`YYYY-MM-DD_任務名稱_摘要.md`
3. **AI 僅記住摘要**：詳細過程已刪除，只保留核心結論（≤200 字）

---

## 摘要檔案結構範例

```markdown
# 任務摘要：[任務名稱]

**完成時間**：YYYY-MM-DD
**結果**：[核心結論，≤200 字]

---

*原始任務已壓縮，詳細過程已刪除*
```

---

**注意**：
- 超過 90 天的任務，AI 會提醒您歸檔到 `ARCHIVE/` 資料夾
- 歸檔後，AI 可以忘記這些任務的細節
