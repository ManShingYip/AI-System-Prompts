# ACTIVE 資料夾

> **用途**：存放進行中的任務（動態記憶，權重 ≥80%）

---

## 使用方式

1. **AI 會自動建立任務檔案**（當用戶提出新任務時）
2. **檔案命名格式**：`YYYY-MM-DD_HH-MM_任務名稱.md`
3. **AI 完整記住**：所有細節都會被記住

---

## 任務檔案結構範例

```markdown
# 任務：[任務名稱]

**提出時間**：YYYY-MM-DD HH:MM
**狀態**：待處理 / 進行中 / 已完成
**用戶原始需求**：[用戶的原話]

## 分析

[任務分析內容]

## 方案選項

[提供的方案]

## 用戶決策

[用戶選擇的方案]

## 執行結果

[執行後的結果]
```

---

**注意**：
- 任務完成後，AI 會將檔案移動到 `COMPLETED/` 資料夾
- 並壓縮為摘要（僅保留核心結論 ≤200 字）
