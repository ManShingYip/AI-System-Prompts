# ARCHIVE 資料夾

> **用途**：存放 90 天以上的舊任務（可遺忘，權重 <10%）

---

## 使用方式

1. **AI 會提醒您歸檔**：當任務超過 90 天後
2. **檔案組織格式**：`YYYY-QX/任務名稱.md`（按季度分類）
3. **AI 可以忘記**：這些任務不再佔用記憶空間

---

## 資料夾結構範例

```
ARCHIVE/
├── 2024-Q3/
│   ├── 2024-07-15_舊任務A.md
│   └── 2024-08-20_舊任務B.md
├── 2024-Q4/
│   └── 2024-10-10_舊任務C.md
└── 2025-Q1/
    └── 2025-01-05_舊任務D.md
```

---

**注意**：
- 歸檔後，任務會從 `WEIGHT_CONFIG.md` 移除
- AI 完全忘記這些任務的存在
- 如需恢復，您可以手動將檔案移回 `ACTIVE/` 或 `COMPLETED/`
